## Example

### Input

```text
https://example.com/search?q=hello world
```

### Encoded

```text
https%3A%2F%2Fexample.com%2Fsearch%3Fq%3Dhello%20world
```

### Decoded

```text
https://example.com/search?q=hello world
```
